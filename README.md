## Description
SkillSphere is a modern e-learning platform designed to connect students with industry experts through high-quality online courses. Users can explore trending courses, discover top instructors, view detailed course information, and manage their learning journey through a secure authentication system and personalized user profiles.

---

## Technologies Used
- Next.js
- React.js
- Tailwind CSS
- DaisyUI
- HeroUI
- Better Auth
- React Hook Form

---

## Key Features

### 1. Secure Authentication System
- Users can sign up and log in using email/password or Google authentication.
- Keeps user accounts and learning data secure.
- Allows users to manage profile and securely logout anytime.

### 2. Course Discovery & Learning Experience
- Users can browse and explore trending courses easily.
- Helps users quickly find relevant learning content.
- Provides detailed course information before enrollment.
- Improves learning experience with a clean and responsive UI.

### 3. Instructor Showcase & Guidance
- Users can explore top instructors and their profiles.
- Helps users choose courses based on instructor expertise and ratings.
- Builds trust by showing instructor credibility and experience.
